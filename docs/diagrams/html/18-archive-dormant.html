03 — Archival & Retirement Flow
Session archive path: At session end, /wrap-up invokes the wrap-up skill which writes a handoff document to LIVE_STATE.json. The session_end.py hook then archives completed handoffs to archive/. New sessions read LIVE_STATE.json on start to resume or start fresh.
  • /wrap-up — creates handoff markdown in LIVE_STATE.json
  • session_end.py — moves handoff to archive/HANDOFF_date_name.md
  • Deprecated components — moved manually to dormant/ subdirectory (agents/, gates/, rules/)
  • Re-activation — move file out of dormant/; remove .dormant suffix from rules files
  • gate_08_temporal.py — retired gate; brute-force scan found not worth the 5s overhead
