04 — Embedding & Search

Embedding Model
  • nomic-embed-text-v2-moe
  • 768-dimensional vectors
  • 8192 token context window
  • Cosine similarity, flat scan
  • Dedup: cosine > 0.85 blocked
  • FNV1a hash for entry IDs
Search Modes
  • semantic — vector cosine, ~30ms
  • keyword — BM25 FTS, ~19ms
  • hybrid — semantic + BM25 fused (default)
  • tags — SQLite tags.db, <2ms
  • observations — scoped to obs table
  • all, code — cross-table and code-focused
